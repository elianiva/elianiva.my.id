---
title: Remove useless LLM comments on code
date: 2025-05-01
type:  til
---

LLM likes to leave useless comments on the code it generates. Use this regex to remove them.

```regex
\/\/[^\S\n]+(?:Add|Remove|Update|Import|Restore).{1,30}(\n)
```

- Two forward slashes
- One or more whitespace non-newline characters
- One of the words "Add", "Remove", "Update", "Import", or "Restore".
- Between 1 and 30 other characters (except newline)
- Newline
