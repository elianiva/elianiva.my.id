We made this bot initially for a more interactive group. It's a simple bot that can do a lot of things. It integrates
with [teknologi-umum/graphene](https://github.com/teknologi-umum/graphene) to provide code snippet image generation.
It can also execute any arbitrary code and return the result thanks to [teknologi-umum/pesto](https://github.com/teknologi-umum/pesto).
